Update: June 4th, 2016

05 June 2016 on Programming, Personal, Update

JUNE 4, 2016

It has been a while since I posted on this blog. Things have been busy, but now I want to try this blogging thing again.

What's new?

Blog:

I migrated this blog over to a Ghost system. The nice folks over at Github provide this really nice student pack. Among the various things that they supply, it also includes servers on Bitnami which allowed me to deploy this very nice looking blog. I will give this a shot.

Toy Language:

This project is still going on. There have been a few changes that I will talk about in a future blog post, but for now, I will just say I have learned a lot since I first started the project.

Work Hard, Play Hard:

I have a renewed drive to develop for the Pebble smartwatch due to recently purchasing a Pebble Round. I have come up with an app that I want to develop which I will speak more about in the future.

Sooooo...

Its a lot of "more to come", but this time hopefully I will stick to this blogging thing. Well that's all for now.
