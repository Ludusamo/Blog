Arduino Libraries: Timer

10 July 2016 on Programming, Arduino, C++, Library, Timer, Github, Open Source

Arduino Libraries

I decided to create my own repository filled with libraries/utilities for Arduino that I would find useful. You can find the repository here. As a first entry, I find myself always implementing a timer in various projects, so I decided to write a standard one.

Timer

To be honest, its not like it takes very long to implement one of these, but I still really shouldn't be wasting time rewriting this anytime I need one. The timer is very simple and will work until the millis() function in Arduino wraps around itself (~50 days according to the Arduino website). The timer automatically starts as soon as it is instantiated and can be reset with a reset() function. If you would like to see further examples and or the source code, feel free to take a look at the library on my Github.
